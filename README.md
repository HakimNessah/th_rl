# th_rl
Pytorch Reinforcement Learning agents in a–Ω iterated prisonner dilemma setting

## Agents
- Discrete:
    - QTable
    - DQN
    - Actor Critic
    - PPO
- Continuous    
    - SAC
    - TD3

## Multi-agent environments
- State-Price
    - Discrete or continuous action spaces
    - Linear price as function of production
- State-Action
    - Discrete or continuous
    - Linear price as function of production

## Buffers
- Standard replay buffer
